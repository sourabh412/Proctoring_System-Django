About BMSCE ISE Proctoring

Students updating records and activities and getting them verified by their proctor is now made so simple, that a student can access their records anytime and anywhere, update them and get them verified by their respective proctor just through this simple web app. While proctors can access the records of students, verify records and grade activity points based the on activity criteria.

The web app behaves differently depending on whether the user is a student, a faculty or an admin, with different features based on the user type. while this web app lets the student to upload files and update records/activities, the proctor has the permission to verify the students' details and access to students' accounts.

This web app is based on Python. It just aggregates all the student information and delivers it to the user when and where required.

This web application is built by students of the ISE dept with support from the ISE dept, BMSCE. Please take time to report any bugs/typos or any other problems you encounter.
